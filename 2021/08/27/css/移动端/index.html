移动端

Posted on 2021-08-27 Edited on 2022-08-14 In css

视口

理想视口

  1. width viewport的宽度
  2. height viewport的高度
  3. device-width 定义输出设备的屏幕可见宽度
  4. initial-scale 初始的缩放比例
  5. minimum-scale 允许用户缩放到的最小比例
  6. maximum-scale 允许用户缩放到的最大比例
  7. user-scalable 用户是否可以手动缩放

移动端特殊样式

  1. box-sizing:border-sizing
  2. -webkit-box-sizing:border-sizing 默认：content-box border-box：margin、padding不会撑开盒子
  3. -webkit-tap-highlight-color:transparent 可以设置高亮颜色以及清除点击高亮
  4. -webkit-appearance:none 清除iOS中预置的样式，如input，botton
  5. -webkit-touch-callout:none 禁用长按弹出菜单

媒体查询

mediaType 媒体类型

| 名称 | 作用 |
| :—:| :—: |
| all | 用于所有设备 |
| print | 用于打印机和打印预览 |
| screen | 用于电脑屏幕，平板电脑，智能手机等。 |
| speech | 应用于屏幕阅读器等发声设备 |

REM 和EM

em

  1. 字体大小相对于父元素字体大小来说的
  2. 其他属性相对于自身的字体大小来说，如果自身没有，则寻找有字体大小的父级元素

rem

  1. 相对于html元素字体大小来说的(通过修改html里面的文字大小来改变页面中元素的大小)
