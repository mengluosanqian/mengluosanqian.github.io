原生js

获取dom有几种方法

  1. getElementsByClassName
  2. getElementById
  3. getElementsByTagName
  4. querySelector
  5. querySelectorAll
  6. getElementsByName

插入DOM

  1. innerHTML
  2. appendChild
  3. insertBefore

点击事件

  1. js事件绑定onclick
  2. html标签绑定事件onclick
  3. 监听事件 addEventListener
    • 事件类型 click、hover等
    • 回调函数
    • 是否阻止冒泡，默认是false，不阻止冒泡

阻止冒泡

  1. stopPropagation
  2. window.event.cancelBubble = true

取消默认行为

  1. preventDefault
  2. window.event.returnValue = false

自定义事件

  1. 使用Event
  2. 使用customEvent （可以传参数）
  3. 使用document.createEvent(‘CustomEvent’)和initEvent() 创建一个事件
  4. document.initEvent： 初始化一个事件
