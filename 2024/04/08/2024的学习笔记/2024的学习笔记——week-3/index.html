打包关键操作

  1. 配置vue.config.js
  2. 添加 copy-webpack-plugin 模块，用于复制文件
    • copyFiles 是复制文件的字段
    • pages 是配置多页面的文件字段
    • configureWebpack 来配置 content.js、background.js 文件
    • css 配置 content.css 文件
  3. 配置 chainWebpack 字段
    • 每次打包生成的 chunk-vendors.js 会跟一个 hash
    • 配置 chainWebpack 字段，对 config 内容进行处理
